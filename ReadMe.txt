===========
How To Use 
===========

1) Text Align
-------------
text-text align value
eg text-center,left,right

2) Position
-----------
pos-position value
eg pos-fixed

3) Padding
----------
To Remove Padding Use pd-direction
To Add Padding Use pd-direction-value

Eg Padding Remove
=================
pd-r pd-left-r

Eg Padding Add
==============
pd-left-20

and same for margin use mr
